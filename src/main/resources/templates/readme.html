1. Descripción y Funcionalidades

La aplicación web del FabLab de Mérida busca ser una plataforma integral para la gestión y difusión de los servicios del laboratorio de fabricación digital de la Universidad de Mérida.

Gestión de Usuarios

Registro, login y perfiles.

Reservas

Calendario y turnos de máquinas.

Cursos

Inscripción y gestión académica.

Administración

Control total de recursos.

3. Instalación

Base de Datos (Docker)
docker network create fablab_net $root = (Get-Location).Path docker run --name fablab-mysql ` --network fablab_net ` -e MYSQL_ROOT_PASSWORD=admin ` -e MYSQL_DATABASE=fablabdb ` -e MYSQL_USER=admin ` -e MYSQL_PASSWORD=admin ` -p 3307:3306 ` -v fablab_mysql_data:/var/lib/mysql ` -v "$root\docker\mysql\init:/docker-entrypoint-initdb.d:ro" ` -d mysql:8.0
Ejecutar Aplicación
./mvnw spring-boot:run

Accede a: http://localhost:8081
