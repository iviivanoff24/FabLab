FabLab Mérida

El futuro se fabrica aquí — plataforma de gestión del laboratorio de fabricación digital.

Resumen

Aplicación web para gestionar usuarios, máquinas, reservas y cursos del FabLab. Incluye interfaz pública, panel administrativo y catálogo de recursos.

Características principales

Usuarios

Registro, inicio de sesión y gestión de suscripciones.

Máquinas y reservas

Catálogo de máquinas, disponibilidad y sistema de reservas con calendario.

Cursos y talleres

Listado de cursos, inscripción y gestión de plazas.

Panel administrativo

CRUD para máquinas, cursos y usuarios.

A futuro

  • Registrar historial de uso por usuario para auditoría.

Estructura del proyecto

  • src/main/java — Código Java (Spring Boot)
  • src/main/resources/templates — Vistas Thymeleaf
  • pom.xml — Dependencias y configuración Maven
  • Documentacion/ — Scripts SQL y notas (MySQL / Docker)
Configuración previa: Base de Datos
Antes de ejecutar la aplicación:

1. Accede a la terminal desde la carpeta general del proyecto ("ProyectoFinal")

2. Ejecuta los siguientes comandos

docker network create fablab_net
$root = (Get-Location).Path
docker run --name fablab-mysql `
  --network fablab_net `
  -e MYSQL_ROOT_PASSWORD=admin `
  -e MYSQL_DATABASE=fablabdb `
  -e MYSQL_USER=admin `
  -e MYSQL_PASSWORD=admin `
  -p 3307:3306 `
  -v fablab_mysql_data:/var/lib/mysql `
  -v "$root\docker\mysql\init:/docker-entrypoint-initdb.d:ro" `
  -d mysql:8.0
Configuración

Ajusta `application.properties` con tu conexión a MySQL (ver carpeta Documentacion/).
